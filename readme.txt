Own text links - own you internal or external links.

== Description ==

You can automatically link keywords and phrases in your posts, pages or custom posts.
Simply select a keywords from your tags or any other keywords you want to make links from your posts or pages.

http://www.youtube.com/watch?v=Ku8vhdHq46w

Features include:

*Work for Custom Post Types
*Process RSS feeds
*Add your Custom Css
*"nofollow" attribute support
*Count views per key
*Expiration per key
*Limit links per page/post
*Select keyword from your tags list.

**Plugin's Official Site**

WPmanage [(http://wpmanage.com/Own-text-links/)](http://wpmanage.com/Own-text-links/)

== Installation ==

This section describes how to install the plugin and get it working.

-    Download the plugin .zip file.
-    Upload the whole plugin folder to your /wp-content/plugins/ folder.
-    Activate the plugin at the plugin administration page.
-    Use the Setting on Own text links menu to make your settings
-    Add new key on Add/edit button in Own text links menu
